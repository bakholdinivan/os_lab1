# Лабораторная работа №1 — Разделение процессов и подсчёт суммы


Этот репозиторий содержит реализацию лабораторной работы по теме взаимодействия процессов через pipe. Решение состоит из двух отдельных программ: родительского процесса (parent) и дочернего (child). Родитель принимает имя файла с командами (каждая строка — набор чисел типа float), запускает дочерний процесс и получает от него результаты через неименованный канал. Дочерний читает строки, суммирует числа в каждой строке и выводит результат в формате `Sum: XX.XX`.

Структура(все внутри solve)
- Makefile
- build/child — исполняемый файл дочернего процесса (после сборки)
- build/parent — исполняемый файл родителя (после сборки)
- parent.c — исходник родителя
- child.c — исходник дочернего
- examples/ — пример входного файла (опционально)

Сборка

make


Запуск
1. Подготовьте текстовый файл с количеством строк, например `input.txt`:
   Пример содержимого:
   1.5 2.3 3.2
   -4.5 15005.0
   10

2. Запустите родителя:
   ./build/parent

   Программа запросит "Enter filename: " — введите имя файла, например:
   test2.txt

3. Родитель создаст дочерний процесс, передаст дочернему stdin, дочерний посчитает суммы по строкам и выведет результаты, которые родитель покажет в консоли:
   Result:
   Sum: 7.00
   Sum: 15000.50
   Sum: 10.00

Ключевые моменты реализации
- Межпроцессное взаимодействие: pipe(), fork/clone(), dup2(), execv().
- Потоки ввода/вывода: родитель перенаправляет файл в stdin дочернего, дочерний перенаправляет stdout в pipe — родитель читает.
- Обработка ошибок: проверка всех системных вызовов (open, pipe, fork, dup2, execv, read, write).
- Формат вывода: числа формируются вручную с двумя знаками после запятой.
